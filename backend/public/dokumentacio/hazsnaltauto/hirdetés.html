Hirdetés feladása

Egy felhasználó hirdetést tud feladni, amely megjelenik a főoldalon.

Ezt egy ilyen csodá űrlapon teheti meg:

Működés

A beérkező adatokat a laravel validate() funkciójával ellenőrizzük. Majd az Eloquent modell használatával a kapott adatokból létrehozzuk a példányt az adatbázisban

$auto = Auto::create([ 'marka' => $request->marka, 'modell' => $request->modell, 'evjarat' => $request->evjarat, 'kilometer' => $request->kilometer, 'meghajtas' => $request->meghajtas, 'allapot' => $request->allapot, 'ar' => $request->ar, 'kep_url' => $imagePath, // Kép URL mentése 'kontakt_info' => $request->kontakt_info, 'leiras' => $request->leiras, 'user_id' => Auth::id(), ]);

A hirdető

A hirdetést feladó felhasználó azonosítóját rögzítjük a hirdetésben, és később ez alapján hivatkozunk rá.

... 'user_id' => Auth::id(), ...
