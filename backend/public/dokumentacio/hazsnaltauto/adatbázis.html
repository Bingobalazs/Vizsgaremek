Adatbázis

Auto

  • id: Egyedi azonosító az autókhoz.

  • user_id: A hirdetést feladó felhasználó ID-ja.

  • marka: Az autó márkája, pl. Toyota.

  • modell: Az autó típusa, pl. Corolla.

  • evjarat: Az autó gyártási éve, egész számként tárolva.

  • kilometer: Az autó futásteljesítménye kilométerben.

  • meghajtas: Az üzemanyag típusa, előre definiált értékekkel:

    • Benzin

    • Dízel

    • Elektromos

    • Hibrid

  • allapot: Az autó állapota, szintén előre definiált értékekkel:

    • Újszerű

    • Jó

    • Használt

    • Sérült

  • ar: Az ár, pénznem formátumban.

  • kep_url: Képek URL-jeinek tárolására szolgáló mező, amely több URL-t is tartalmazhat (pl. JSON formátumban).

  • kontakt_info: Hirdető elérhetősége (pl. telefonszám vagy email-cím).

  • leiras: Szöveges mező az autóról szóló részletek számára.

Felhasználók

  • id Felhasználó egyedi azonosítúja, másol: user_id

  • name Felhasználó neve

  • email Regisztrációkor megadott elektronikus postafiók

  • email_verified_at Az e-mail hitelesítésének időpontja

  • password Felhasználó jelszava

  • remember_token Fingom sincs ez mi, Csabika bizti tudja

  • created_at Felhasználó létrehozásának ideje

  • updated_at Adatok legutóbbi módosítása

  • code Elektronikus postafiókot hitelesítő kód
